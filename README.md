# 🎬 AnimeList Platform

A simple full-stack Anime listing platform built with Next.js (frontend), Node.js + Express (backend), Prisma ORM, PostgreSQL, and Docker.

---

## 🚀 Tech Stack

**Frontend**
- Next.js (App Router)
- TypeScript

**Backend**
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL

**DevOps**
- Docker
- Docker Compose
